1.4 - Les modules

Les modules servent à définir l'ensemble des outils/componsant définit est utilisé. Les modules permettent de segmenter l'application en sous partie est ainsi optimisé le site. Si le mode lazy loading est utilisé cela permet de ne loader (instancier les services, importer les composant, crées les routes) que lorsque celui ci est appeller par l'utilisateur.
