2 - Fonctionalités

Ici nous allons voir les divers fonctionalités angular. Dans un premier temps parlons des modules. Un module est un fichier en .module.ts (convention de nommage), qui référence tous les composants/services crées ainsi que toutes les librairies utilisées par ces composant. Il permet de déclarer également quel objet nous voulons exporter vers d'autres modules. Pour résumer, un module est la notice du composant. Nous reviendrons plus en détail sur la partie module.
